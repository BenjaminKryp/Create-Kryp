📢 Update v0.2.3r

⚠️ Important Notice

📝 About This Update

Abundance of fixes and changes from updating some mods.

Big Faunify update with many fixes, new stuff, and remodels for most mobs, plus a couple of new ones.


Fixed caves generating really small and ravines and cave entrances being really rare.

Fixed crash with Waystones.

Fixed lag and log spam when looking at some entities (full fix hopefully in a future update) and oil generation lag.

Fixed opossums being able to be heard from really far away with no sound drop-off.

Fixed pixie particles being visible even when the pixie itself was not in render distance.

Fixed snow spawning in tall piles and causing a lot of lag when generated, even in wrong seasons.

Fixed some blocks having no sound when you walk on them.

Fixed some entity sounds being heard as long as the entity is rendered with no sound drop-off if you walk further away.

Fixed some memory leaks.

Fixed version being wrong again.

Fixed villages generating weirdly and being really rare, along with other structures.

Made it so you can see other players on the map no matter where they are.

Reduced memory usage a bit.

Removed the 90% RAM usage message in chat when RAM usage reaches 90%.

Some blocks/ores/mobs/biomes/etc names are better. I accidentally put [The Rename Compat Project](https://www.curseforge.com/minecraft/texture-packs/the-rcp) in a datapack instead of a resource pack, so now they are more mod compatible and don't share the same names if they are slightly different.

Some modded armor trims made armor look glitched because [Armor Trim Compat](https://www.curseforge.com/minecraft/texture-packs/armor-trim-compats) was put in a datapack instead of a resource pack.
