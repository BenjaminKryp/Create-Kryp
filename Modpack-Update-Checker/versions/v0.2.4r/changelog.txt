📢 Update v0.2.4r

⚠️ Important Notice

📝 About This Update

Added tier upgrade support for more Quark chest types.

Fixed game becoming unresponsive if you die on a server.

If a world/server is lagging it should not look or feel as laggy anymore hopefully.

Made it so you no longer see other players’ icons in the world. It was supposed to only show on the map, now it does.

Potentially fixed lower FPS when not holding anything in the main hand while cosmetics from the Essential Mod are enabled.

Potentially improved world generation speed on slower CPUs.

Tweaked some shader settings a bit.
